# 🎬 豆瓣电影深度挖掘 · Douban-CineMatrix

基于豆瓣 Top250 电影的硬核数据分析与自然语言处理项目。通过爬虫获取电影信息及用户短评，结合 LLM 进行情感分析，并生成交互式可视化仪表盘。

## ✨ 主要功能

- 🕷️ **Top250 电影信息爬取**：断点续抓、随机 UA、Cookie 模拟，获取电影名称、导演、年份、评分、类型、评价人数等。
- 🎥 **导演权力矩阵**：以气泡图展示上榜电影数量与平均评分的关系，识别“神级导演”。
- 🌀 **年代题材旭日图**：按十年年代 → 题材类型 → 具体电影逐层下钻，颜色代表评分，大小代表热度。
- 🧠 **短评情感分析（LLM）**：调用大模型（DeepSeek / OpenAI 兼容接口）对短评进行正面得分、争议判断、关键词提取。
- 📊 **口碑对比仪表盘**：多维度对比不同电影的情感分布、争议比例、差评词云等。
- ☁️ **差评词云**：自动提取差评核心词汇，生成词云图片。
